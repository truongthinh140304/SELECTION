Privacy Policy



Selection Translate \& Wiki helps users translate selected text on web pages and look up related information from Wikipedia.



Data processed:

\- Selected text from web pages

\- Optional page title or page URL if the user saves a translation to history

\- User preferences and saved translation history stored locally in the browser



How data is used:

Selected text may be sent to third-party APIs such as Wikipedia, MyMemory Translation, and Google Generative Language API to provide translation and reference results. The extension does not sell user data.



Data storage:

Saved translations and preferences are stored locally in the user's browser using browser storage.



Data sharing:

The extension only sends selected text to external APIs when needed to provide translation or information lookup features. The extension does not use user data for advertising, creditworthiness, or unrelated purposes.
